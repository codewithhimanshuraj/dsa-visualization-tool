# 🚀 DSA Visualizer

An interactive Data Structures & Algorithms Visualizer built to help students understand algorithms through animations and real-time execution.

---

## 🌐 Live Demo
👉 https://dsa-visualization-three.vercel.app/

---

## ✨ Features
- 📊 Algorithm visualization
- ⚡ Real-time animations
- 🎯 Sorting & Searching algorithms
- 🎨 Modern UI
- 🌙 Dark/Light mode

---

## 🛠️ Tech Stack
- Next.js
- TypeScript / JavaScript
- Tailwind CSS
- Framer Motion
