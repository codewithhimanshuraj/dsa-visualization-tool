🚀 DSA Visualizer

An interactive Data Structures and Algorithms Visualizer built to help students and developers understand complex concepts through animations and real-time execution.

🌟 Features
📊 Visualize popular algorithms step-by-step
🎯 Supports multiple data structures
⚡ Real-time animations for better understanding
🎨 Clean and modern UI
🌙 Dark/Light mode support
🧠 Beginner-friendly learning tool
🧩 Algorithms Included
🔹 Sorting Algorithms
Bubble Sort
Selection Sort
Insertion Sort
Merge Sort
Quick Sort
🔹 Searching Algorithms
Linear Search
Binary Search
🔹 Graph Algorithms (optional if added)
BFS (Breadth First Search)
DFS (Depth First Search)
Dijkstra’s Algorithm
🔹 Data Structures (optional if added)
Stack
Queue
Linked List
Tree
🛠️ Tech Stack
Frontend: React / Next.js
Language: TypeScript / JavaScript
Styling: Tailwind CSS
UI Components: shadcn/ui
Animations: Framer Motion
