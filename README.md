🚀 DSA Visualizer

An interactive Data Structures and Algorithms Visualizer built using modern web technologies to help students and developers understand algorithms through beautiful animations and real-time execution.

🌐 Live Demo

👉 https://dsa-visualization-three.vercel.app/

✨ Features
🎯 Step-by-step algorithm visualization
⚡ Smooth animations with real-time updates
📊 Multiple sorting & searching algorithms
🎨 Modern and responsive UI
🌙 Dark / Light mode support
🧠 Beginner-friendly learning experience
🧩 Algorithms Supported
🔹 Sorting Algorithms
Bubble Sort
Selection Sort
Insertion Sort
Merge Sort
Quick Sort
🔹 Searching Algorithms
Linear Search
Binary Search
🔹 (Optional / Future)
Graph Algorithms (BFS, DFS)
Data Structures Visualization
Stack
Queue
Linked List

⚙️ Getting Started
1️⃣ Clone the repository
git clone https://github.com/codewithhimanshuraj/dsa-visualization-tool.git
2️⃣ Navigate to project folder
cd dsa-visualization-tool
3️⃣ Install dependencies
npm install
4️⃣ Run development server
npm run dev
5️⃣ Open in browser
http://localhost:3000
🚀 Deployment

This project is deployed using Vercel.

To deploy your own version:

Push project to GitHub
Import repository in Vercel
Click Deploy
